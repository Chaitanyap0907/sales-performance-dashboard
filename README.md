# sales-performance-dashboard
"Interactive Power BI dashboard to analyze sales performance, profitability, and discount impact across regions and product categories.”

## Project Overview
This project analyzes sales and profit data using Power BI to identify trends and business insights.

## Tools Used
- Power BI
- Excel

## Dataset
The dataset contains sales transactions including region, category, sales, profit, and discount.

## Key Insights
- Higher discounts negatively impact profit margins
- Some product categories generate high sales but low profit
- Sales performance varies across regions

## Files Included
- dashboard.pbix → Power BI dashboard file
- sales_dataset.csv → Dataset used
- dashboard_screenshot.png → Dashboard preview

## How to Use
Download the .pbix file and open it in Power BI to explore the dashboard.
